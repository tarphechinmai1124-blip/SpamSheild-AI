# 🛡️ SpamShield AI

A Machine Learning-based Spam Detection Web App that classifies messages as **Spam** or **Not Spam** in real-time.

---

## 🚀 Live Demo
🔗 https://spamsheild-ai-j2kkuprfqkpm34af3bdfay.streamlit.app/

---

## ✨ Features
- 📩 Real-time spam message detection
- 🧠 Machine Learning model (NLP-based)
- 📊 Clean and interactive UI using Streamlit
- ⚡ Fast predictions
- 🎯 High accuracy classification

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

---

## 📌 How It Works
1. User enters a message
2. Text is transformed using a trained vectorizer
3. ML model predicts whether it's spam or not
4. Result is displayed instantly
